Projects Module (Portfolio)

The new Projects section allows you to document technical or creative projects in detail.

Structure

  • Basics: Title, Pitch, Difficulty, Duration.
  • Team: Credits and contributors.
  • Things: Lists of Hardware components and Software apps.
  • Story: Full description with rich text and images.
  • Attachments: Links to code repositories (GitHub) or schematics.

Frontend: Projects are displayed in a dedicated gallery (projects.php) with filters by difficulty (Easy, Intermediate, Advanced, Expert).

Chat System (WhatsApp-Style)

A fully featured instant messaging system integrated into the site.

Key Features

  • Real-time: Instant message delivery via Ajax polling.
  • Rich Media: Send images via Drag & Drop or file selection.
  • Emojis: Integrated modern emoji picker.
  • Status (Stories): Post ephemeral updates visible for 24 hours.
  • Organization: Archive conversations and Star important messages.
  • Indicators: "Typing..." status, Online/Offline status, Read receipts (Blue ticks).

File Synchronization Tool

Located in Admin > Files > Sync, this tool reconciles the database with the actual files on your server.

How it works

  • Import: Scans the /uploads/files/ directory. If a file exists on disk (uploaded via FTP) but not in the database, it is automatically added to the Media Library.
  • Cleanup: Checks the database against the disk. If a file entry exists in the DB but is missing on the server, the entry is removed to prevent broken images.

Installation

  1. Upload all files to your server (Ensure vendor/ folder is present).
  2. Import database.sql into your MySQL database.
  3. Rename includes/db.php.example to includes/db.php and configure DB credentials.
  4. Set Write Permissions (755/777) on uploads/ folders.
  5. Login at /admin (Default: admin@admin.com / password).

Automated RSS Import

Set up a Cron Job to fetch feeds automatically:

wget -q -O - "https://YOUR-SITE.COM/admin/run_rss_import.php?key=YOUR_SECRET_KEY" >/dev/null 2>&1
